# MERN Stack Real Time Chat App

## Tech Stack

- MongoDB
- Express
- React
- Node.js
- Socket.IO
- TailwindCSS
- daisyUI

## Key Features

- Authentication && Authorization with JWT
- Real time messaging with Socket.IO
- Online user status (Socket.IO and React Context)
- Error handling both on the server and on the client

## Usage

### Setup ```.env``` file

```js
PORT=...
MONGO_DB_URI=...
JWT_SECRET=...
NODE_ENV=...
```

### Build the App

```shell
npm run build
```

### Start the App

```shell
npm start
```
